What is Git?

Software version control system


Source code management


Distributed - many developers can work on a given project from different locations and networks

Why Git?

Extremely fast and efficient


Lightweight


Largely local-based, not dependent on a remote network


Easy to learn

…And Then There is GitHub

https://github.com/


Web-based hosting of Git projects


Fosters even greater collaboration!


Social

Getting Started

RTFM @ http://help.github.com/ (you will need to download and install Git first)

The Most Important Commands

git clone - Download a copy of a repository


git pull - Update your repository with latest changes from remote


git add - Add file(s) to your staging area


git commit - Record your snapshot into your history; "bless" your work


git push - Upload your changes to the remote repository (e.g., GitHub)
